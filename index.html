WEB DEVELOPMENT

Web development is the comprehensive process of creating, building, and maintaining websites and web applications
that run on a browser. It encompasses a wide range of tasks, from the initial planning and design to the actual coding, deployment, and ongoing maintenance.

Front End Development

Front-end development is the practice of building the visual and interactive parts of a website or application that users
see and interact with directly in their web browser. It is also known as "client-side" development and focuses on
creating a smooth, engaging, and optimized user experience (UX) and user interface (UI)

Back End Development

Back-end development focuses on the server-side logic, databases, and infrastructure that power the functionality of
websites and applications, working behind the scenes to process requests, manage data, and ensure security and performance.
Users do not directly interact with the back-end, but it is essential for dynamic and data-driven applications.

Database

A database is an organized, electronic collection of structured data, like records, files, or media, designed for efficient
storage, management, and retrieval, usually controlled by a Database Management System (DBMS) and often using SQL for interaction, making it central for applications needing quick
access to information, from customer lists to product inventory. Think of a library (database) holding books (data) for easy searching and borrowing.
